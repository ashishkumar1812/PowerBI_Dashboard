📊 Sales Data Analysis Dashboard (Power BI)
📌 Project Overview

This project focuses on analyzing a sales/retail business dataset using Power BI. It demonstrates the complete data analysis workflow — from data cleaning to dashboard creation — to extract meaningful business insights.

🎯 Objectives
- Analyze overall sales performance
- Track customer purchasing behavior
- Identify top-performing products and categories
- Understand regional sales trends
- Calculate key metrics like total sales, tax, and revenue

📂 Dataset Description
The dataset contains transaction-level sales data, including:
- Order ID
- Product & Category
- Sales Amount
- Tax Amount
- Quantity
- Customer Details
- Region / Location
- Order Date

🛠️ Tools & Technologies Used
- Power BI – Data visualization & dashboard creation
- Power Query – Data cleaning & transformation
- DAX – Calculated columns and measures
- 
📊 Key Features
- Interactive dashboard with filters & slicers
- Sales trends over time
- Category-wise and region-wise analysis
- KPI indicators for business performance
  
🧮 Sample DAX Calculation
- TotalSalesAmount = Sales[SalesAmount] + Sales[TaxAmt]
- 
📈 Insights Generated
- Monthly and yearly sales trends
- Best-selling products and categories
- High-performing regions
- Revenue contribution analysis
-

Data Collection → Data Cleaning → Data Modeling → DAX Calculations → Dashboard Creation
